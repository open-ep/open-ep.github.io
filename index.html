We Make the Hardware.
You Own the Code.

Open-EP designs high-quality, custom e-paper modules and provides 100% open-source drivers (even supporting Linux Kernel upstream) and software. No proprietary black boxes. Just pure, reflective pixels.

Furthermore, for software demos, we select world-renowned development boards, single board computers, box PCs, or Industrial PCs, aiming to cover as many applications as possible! We also welcome new contributors to join us!
